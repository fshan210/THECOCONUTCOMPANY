## Approved sourcing policy

If future assets are needed, use only:

- Project-owned/local assets.
- Unsplash.
- Pexels.
- Openverse.
- Wikimedia Commons.
- Public-domain SVGs.

Every downloaded asset must include:

- Source URL.
- Author/creator if available.
- License.
- Date downloaded.
- Destination path.
- Usage notes.

Do not use Firebase Storage for Phase 4.5 motion assets.
